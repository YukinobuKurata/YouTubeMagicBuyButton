YouTubeMagicBuyButton
=====================

「YouTube動画があらかじめ設定した時間再生された後に購入リンクを表示する仕組み（HTML、JavaScript、CSS） 」<br />
&lt;iframe&gt;タグの埋め込みコードをつかったHTML5形式の動画プレーヤーです。
